# Price Scraper — парсер цен

Скрипт на Python, который собирает названия, цены, наличие и рейтинг
товаров с сайта и сохраняет их в CSV-файл (открывается в Excel/Google Sheets).

Демо работает на [books.toscrape.com](http://books.toscrape.com) — сайте,
специально созданном для тренировки парсинга (в отличие от реальных
магазинов, тут можно собирать данные без ограничений и вопросов этики).

## Что делает скрипт

1. Открывает страницы каталога одну за другой.
2. С каждой страницы забирает: название книги, цену, наличие на складе, рейтинг (1-5).
3. Сохраняет всё в файл `books.csv`.
4. Делает паузу 1 секунда между страницами, чтобы не перегружать сайт запросами.

## Установка и запуск

```bash
cd price-scraper
python3 -m venv venv
source venv/bin/activate   # на Windows: venv\Scripts\activate
pip install -r requirements.txt
```

Запуск (соберёт первые 5 страниц каталога по умолчанию):

```bash
python3 scraper.py
```

Собрать больше или меньше страниц:

```bash
python3 scraper.py --pages 10
```

Сохранить в файл с другим именем:

```bash
python3 scraper.py --pages 10 --output my_books.csv
```

Результат — файл `books.csv` (или указанный тобой) в этой же папке.
Столбцы: `title`, `price_gbp`, `availability`, `rating`.

## Как адаптировать под реальный сайт

Этот скрипт — шаблон. Чтобы собирать данные с другого сайта:

1. Открой страницу нужного сайта, найди в HTML-коде (через "Просмотр кода
   страницы" в браузере) селекторы для нужных данных (название, цена и т.д.).
2. Замени `BASE_URL` и селекторы в функции `parse_page` под структуру этого сайта.

**Перед тем как парсить реальный сайт:**
- Проверь файл `сайт.ру/robots.txt` — там написано, что сайт разрешает собирать автоматически, а что нет.
- Проверь пользовательское соглашение сайта (Terms of Service) на предмет запрета скрейпинга.
- Не убирай паузы между запросами (`time.sleep`) — это не только вежливо, но и снижает риск блокировки твоего IP.
